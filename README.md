# 🎵 Music Library Manager

A Python project built while learning core data structures.

## 📌 About
This project was built as part of my Python learning journey. 
It demonstrates the use of core Python data structures to 
manage a music album library.

## 🛠️ Concepts Used
- **Dictionaries** — store album info as key-value pairs
- **Lists** — manage a mutable playlist
- **Tuples** — store fixed genre categories
- **Sets** — track unique favourite artists
- **CSV** — save and export library data

## 🚀 How to Run
1. Make sure Python is installed on your computer
2. Clone this repository
3. Run the file:

python music_library.py

## 📂 Files
- `music_library.py` — main Python script
- `albums.csv` — generated CSV file of album data
- `README.md` — project documentation
